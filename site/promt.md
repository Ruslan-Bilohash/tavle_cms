**Структура проекту (створи всі файли повністю, без обрізання коду):**

1. `index.php` — головна сторінка з пошуком, фільтрами, слайдером, картками авто.

2. `car.php?id=1` — детальна сторінка оголошення.

3. `admin/` — повна адмін-панель (dashboard, CRUD для брендів/авто/користувачів/новин).

4. `config.php`, `functions.php`, `db.php`.

5. Папки: assets/css (Tailwind або custom), assets/js, uploads/cars, languages/*.php.

6. `htaccess` для ЧПУ.

7. SQL dump з таблицями + демо-даними.
