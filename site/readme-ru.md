## Основные возможности

### Публичный сайт
- **Объявления авто** — фильтры, AJAX-каталог, карусель фото
- **Номера** и **спецтехника** — отдельные разделы
- **Мастер добавления** — 4 шага: контакты → данные → фото → предпросмотр
- **Черновики** — сохранить без публикации
- **Массовая загрузка фото** — до 20 изображений, drag & drop
- **Аккаунты** — создаются автоматически при первом объявлении
- **Мои объявления** — редактирование своих объявлений

### SEO
- `hreflang`, canonical, Open Graph, Schema.org
- SEO-лендинги марка/модель, региональные страницы
- `llms.txt` для AI-агентов

### Админ-панель
- Dashboard со статистикой и черновиками
- Объявления, бренды, модели, пользователи, новости
- Навигация и настройки (SEO, валюта, cookies)
- Загрузка фото в объявлениях
- 4 языка интерфейса админки

## Технологии

PHP 8+, SQLite, Bootstrap 5 (админка), собственный CSS/JS (фронт).

## Установка

```text
/tavle/site/
  data/            ← права на запись
  uploads/cars/    ← права на запись
  install.php      ← проверка (удалить после деплоя!)
```

1. Загрузить файлы на сервер.
2. `chmod 755` для `data/` и `uploads/cars/`.
3. Открыть `install.php` — БД создаётся автоматически.
4. Войти в `/admin/`, сменить пароль admin.
5. Удалить `install.php`.
6. `APP_ENV = production` в `config.php`.

## Публикация объявления

1. `add.php` → контакты (email создаёт аккаунт).
2. Данные авто + фото.
3. **Опубликовать** или **Сохранить черновик**.
4. Редактирование: `my-listings.php` или `add.php?edit={id}`.
